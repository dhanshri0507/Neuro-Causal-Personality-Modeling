# MBTI Dataset Sample

The dataset used in this project is the MBTI dataset from Kaggle. Due to size constraints, the full dataset and the complete processed features (including files like `phase2_features_backup_random_J.npz` and full `phase1_cleaned.json`) are not included in this repository.

A small sample is provided here in `Dataset/Sample/` for demonstration purposes.

### Full Dataset

To run the complete training pipeline, please download the full dataset from:
[Kaggle MBTI Type Dataset](https://www.kaggle.com/datasets/datasnaek/mbti-type)

Place the downloaded `mbti_1.csv` inside `../data/raw/` (create the directories if needed), and then run the data preprocessing scripts as described in the main repository README.
